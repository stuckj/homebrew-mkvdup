# Homebrew Tap for mkvdup

This is the official Homebrew tap for [mkvdup](https://github.com/stuckj/mkvdup), a storage deduplication tool for MKV files and their source media.

## Installation

```bash
brew tap stuckj/mkvdup
brew install mkvdup
```

## Canary (Pre-release) Version

The canary version provides early access to new features and can be installed alongside the stable version:

```bash
brew tap stuckj/mkvdup
brew install mkvdup-canary
```

## Updating

```bash
brew update
brew upgrade mkvdup
```
